## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Only vulnerabilities related to the `*.puku.sh` domain scope should be reported via email [security@puku.sh](mailto:security@puku.sh).

Please email us at  with details and steps to reproduce.

## Scope

The following assets are in scope for security testing and responsible disclosure:

* `*.puku.sh`
* Puku Editor desktop binaries:

  * Windows
  * Linux
  * macOS

Download binaries from:

[Puku Downloads](https://puku.sh/download)

## Disclosure Guidelines

We commit to acknowledging vulnerability reports immediately, and will work to fix active vulnerabilities as soon as we can. Critical incidents will be communicated via email to all affected users.

We appreciate your help in making [Puku Editor](https://puku.sh) more secure for everyone. Thank you for your support and responsible disclosure.
